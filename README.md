# CODSOFT_Task-1

## This project demonstrates the fundamentals of:

* Natural Language Processing (NLP) — keyword detection
* Pattern Matching — Matching user input against rule sets
* Conversation Flow — Maintaining a chat interface


## Working:
This chatbot works using if-else conditions and pattern matching.
First it takes user input, converts it into lowercase, and checks for specific keywords like “hello”, “help”, or “bye”. Based on the matched keyword, the chatbot returns a response. If no keyword matches, it displays a default message saying it cannot understand the query. 
This helps in understanding the basic flow of conversation and simple natural language processing.
